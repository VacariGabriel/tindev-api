## About
This API is part of the project. [Tindev Front](https://github.com/VacariGabriel/tindev-front)

## Install project

**Without Docker**

`cd tindev-api/`

`npm install`

`npm start`

**With Docker**

`docker build -t tindev-api .`

`docker run -p 3333:3333 tindev-api`
